[주요 마크업]
   - 마크업 형태 종류
    - <시작마크업> </끝마크업>
    - <마크업명 />

1.<html> </html> : html 시작과 html 끝을 알리는 마크업
2.<head> </head> : html 문서의 다양한 옵션정보 읽어드리는 마크업
3. <title> </title> : html 문서의 제목
4. <body> </body> : html 문서의 * 본문 *

5. <h1> </h1> : h1~h6까지 제목 마크업, 글자크기/굵게
6. <p> </p> : 문단 마크업, 구분/여백 제공
7. <br/> : 줄바꿈
8. <hr/> : 가로 구분선 마크업
9. <ul> </ul> : unoder list 순서없는 목록
10. <ol> </ol> : oder list 순서있는 목록
11. <li> </li> : 목록의 항목

12.<img src=""/> : 이미지 마크업
    - 속성 : src = "이미지 파일의 경로"
    - 경로기준 종류
        1. 절대경로 :
        2. 상대경로 :
